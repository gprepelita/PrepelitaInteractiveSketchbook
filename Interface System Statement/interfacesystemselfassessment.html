Interface System Self-Assessment

Criteria

Navigation Model & System Coherence

Score: 19
Justification: My interface has the same navigation structure across all pages, both mobile and desktop. The user gets to each page the same way and the home button is present on all pages for ease of navigation. Active states could be improved.

Hierarchy & Mobile-First Logic

Score: 20
Justification: The desktop version uses a grid to display all the pages in the sketchbook, utilizing the extra space given over the mobile version. The mobile version adapts the layout to better accommodate scrolling.

Explain how hierarchy survives compression and expands across contexts

Score: 13
Justification: Hierarchy survives compression through smaller text and button sizes in the mobile version. There may be some unclear cues or system status that I think I can improve through hover states.

Theoretical Integration (Norman / Nielsen / Shneiderman)

Score: 14
Justification: Main heuristics applied are minimalist design, match between system and real world, consistency and standards, and recognition rather than recall. Four golden rules I included are consistency, universal usability, closure, and keeping users in control. I discuss these in my interface system statement.

Cross-Page Consistency

Score: 10
Justification: All the pages in my interace are consistent in their design and functionality, as well as the way elements are structured, such as the placement of type and buttons. There is also the consistent banner at the top of each page with the title of the page.

Risk & Structural Ambition

Score: 3
Justification: I went with a very simple and minimal approach to my interface, and kept it pretty safe since I'm not completely comfortable with Figma yet and am still learning all the ins and outs.

Iterative Development

Score: 10
Justification: While the basic idea for the design remained mostly the same, I did make meaningful changes with each iteration, which is visible in the screens. Over the iterations, I added the filter button, home buttons and search bar, adjusted the micro project page, and made some changes to the layouts.

Participation & Critique Engagement
