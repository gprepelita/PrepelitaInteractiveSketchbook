Week 2 Prototype

Structural Reflection

This week I created the mobile interface wireframes for the home page, the micro project page and a template for the p5.js sketch pages. For the home page, I iterated on the previous week's version and changed the sizing of the boxes that would contain the navigation to each other page. I made the boxes much narrower so that more could fit on the screen, leading to less scrolling for the user. I also added a search bar for convenience, as well as a filter button that would allow the user to filter and only show certain types of projects (e.g., sketches, reflections, micro project iterations, etc.). For the micro project page, I wanted the screen to resemble the main menu that the user sees when they open the micro project. I kept all the text the same with the exception of the bottom instructions. For the mobile version, instead of saying "Click to Start" like it normally does, it would say "Flip Phone to Start" since my micro project is a game best played on a wide screen. I also included a basic landscape version of the main game screen that would appear once the phone is flipped. Finally, for the sketch page template, I kept a minimal design with the title of the sketch at the top, matching the home page, the description of the sketch below that, and the sketch itself taking up the rest of the screen below that.
